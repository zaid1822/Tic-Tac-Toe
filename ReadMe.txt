Tic-Tac-Toe Game using Javascript by Mohammed Zaid
-------------------------------------------------------------------------------------------------------------------------------------
Open "main.html" file in Google Chrome to play the game.

First page is of Score-Board. 
Scores are saved on localStorage and therefore will not be lost after reloading or closing the browser.

Click on 'Start' button to start the game.

The rules are same as standard Tic-Tac-Toe game.

Press 'RESET' button to reset game.

Press 'SCORE' button to see scores.

Enjoy!!
